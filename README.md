# 📌 Overview

This project presents an interactive Shoe Sales Dashboard built using Power BI, designed to analyze sales performance across brands, countries, product types, and sales channels.

The goal of this analysis is to uncover actionable business insights that can help improve revenue, optimize product strategy, and identify high-performing markets.

# 🎯 Objectives
Analyze overall sales performance
Identify top-performing brands and products
Evaluate sales channels effectiveness
Understand regional (country-wise) trends
Explore seasonal patterns in sales
Provide data-driven insights for decision-making
